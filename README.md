# tutoriel_video_remotion

# 🎬 Mission Développeur Front-End : Vidéo Interactive avec Remotion

Bienvenue sur le dépôt officiel de la mission **Vidéo Interactive** pour notre SaaS **Content-SEO.ai**. 👋

## 📋 Contexte de la mission

Pour le lancement de notre plateforme, nous souhaitons dynamiser notre Landing Page (développée avec React/Antigravity) avec une vidéo d'introduction percutante. 

Plutôt que d'utiliser un logiciel de montage classique (comme Premiere Pro), nous allons générer cette vidéo **entièrement via du code React** en utilisant la librairie [Remotion](https://www.remotion.dev/). 

**Votre objectif :** Créer une animation fluide de 15 secondes qui simule l'interface de notre SaaS et l'action de notre Intelligence Artificielle, afin d'inciter les visiteurs à s'inscrire.

---

## 🎯 Les Livrables Attendus

1. **Le Code Source :** Un projet Remotion propre, commenté et fonctionnel.
2. **La Vidéo Finale :** L'export final en format `.mp4` (1080p, 30fps).
3. **L'Intégration (Bonus) :** Une PR (Pull Request) sur le dépôt principal de la Landing Page intégrant la vidéo en lecture automatique (`autoPlay`, `loop`, `muted`).

## 🚀 Comment démarrer ?

1. Clonez ce dépôt sur votre machine locale :
   ```bash
